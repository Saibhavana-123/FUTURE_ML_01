
Sales Forecasting using Machine Learning

📖 Project Description

This project focuses on predicting future sales using historical data from a retail dataset. By analyzing past sales trends, the system forecasts future demand, helping businesses make better decisions related to inventory, staffing, and financial planning.

🎯 Objective
Predict future sales based on past data
Analyze trends and patterns in sales
Provide insights for business decision-making
📁 Dataset

The dataset used is the Superstore Sales Dataset.

It includes:
Order Date
Sales
Customer Details
Product Information
Region and Location Data
Key features used:
Order Date (for time analysis)
Sales (target variable)

🛠️ Tools and Technologies
Python
Pandas – Data processing
Matplotlib – Data visualization
Scikit-learn – Machine Learning model
Visual Studio Code – Development environment

⚙️ Project Workflow
1.Load dataset
2.Clean and preprocess data
3.Convert date column into datetime format
4.Create time-based features (Year, Month, Day)
5.Aggregate sales by date
6.Visualize sales trends
7.Train Linear Regression model
8.Predict future sales
9.Evaluate model performance using MAE

📊 Output
Sales trend graph
Future sales forecast graph
Model accuracy (MAE value)

🤖 Model Used

Linear Regression

The model predicts sales based on time (date converted into numerical format).

📉 Evaluation
Metric used: Mean Absolute Error (MAE)
Lower MAE indicates better prediction accuracy
💼 Business Applications
📦 Inventory planning
👨‍💼 Staff management
💰 Budget forecasting
📉 Reducing overstock and losses
✅ Conclusion

This project demonstrates how machine learning can be used to forecast sales and support business decisions. By analyzing historical data, businesses can plan more efficiently and reduce risks.
